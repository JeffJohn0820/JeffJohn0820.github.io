宇宙之大

Jul 5, 2019

【最近写小说不知道为什么总是处在男性视角，这一篇也算是写作课作业。怎么说，写完觉得不尽如人意，但是好歹还是写完了的。至于标题嘛，原谅这个标题废爱许嵩爱得深沉。】

诶，兄弟，在这呐，来来来，快过来坐。哎哟你说这天也怪冷的，大晚上还把你叫出来真是不好意思，那什么，老板，再拿两瓶啤酒……嗯？别呀，今儿个找你出来就是我请客，别客气……哦，弟妹交代了是吧？也行也行，反正我自己喝着也是一样的……什么事？其实吧，也没什么事。诶，对了，你跟弟妹，是第几次谈恋爱啊？……哦，都是初恋对吧？小子，可得好好珍惜呀。唉，说起来，今天把你叫出来，也跟我初恋有点关系。就是这样的，前天，不是月初嘛，我就翻那个信箱拿银行账单，结果你猜怎么着？嘿，给我翻出这玩意……对，没错，就是结婚请柬……当时吧，我没敢翻开看，可是我认得信封上的字。她写我名字的时候，那个“宇”字的宝盖头是连着下边的“于”的……就是啊，跟你讲讲她的事。

我俩高中三年都在一起。高考前几次模拟，我们俩成绩差得不多，考到同一所大学没什么问题的。可是分数出来之后她爸她妈商量着就把她志愿改了，说是那个学校专业好。等拿通知书那天我俩大眼瞪小眼，都没说话。回家之后据说她跟她爸妈闹了一场，然后跟我打电话说分手。她说异地恋不会有好结果的，她不想耽误我。我也没什么反应，顺口就应了……对，你说对了，她不是想分手，她想的是让我挽留。你说女人啊，怎么这么奇怪呢？想要什么偏不直说，就是要拐个弯，让你猜了又猜，还那么口是心非……没分呐。过了三天她又打电话给我，哭着骂我“混蛋”“没良心”，幸好我还没木到那地步，赶紧哄着，好说歹说哄了回来……诶老板，我要的羊肉串呢？！

上了大学之后也就那样，就是更进了一步，该发生的都发生了。有句话我觉得还是挺有道理的，就是女人吧，是因性而爱的。她觉得她是我的人了，她就不会离开我了，她要对我忠贞……没错，谢谢老祖宗吧，也得谢谢她是个书呆子，要是什么现代新新女性，我指不定头上多少绿呢。好了不开玩笑了，继续说她。我们俩一个南下一个北上，虽说都没走出浙江吧，可还是远，我吧又懒，学校事情一多就不怎么顾得上她，就是每天例行公事打个电话，说两句今天干了什么事就挂了。可能是觉得我太敷衍，有一天她给我打电话的时候终于爆了，她问我是不是看上别人了。我说她别闹了，她更不依不饶，然后那个星期五她连夜乘动车过来，星期六天还没亮站到我寝室楼下叫我名字——这是她第二次跟我提分手。当然，我也知道，这么远过来找我肯定不单是说分手，不，她就不是为了分手来的。兄弟啊，你记住了，一个女人，她如果对你闹，骂你打你，那是她还在乎你，真的是在乎你……哦？弟妹的电话？要过来吗？……行啊，别拦着，她肯定是不放心你在外边，这也是在乎你呐……能啊，来就来吧，让她来吧。

那个周末，我陪着她在大学城附近玩，她挨个认了我们班女生，也没多少，就四个——把她们跟QQ啊，微信啊对应起来才算心满意足地回去了。女人都说这叫防微杜渐，我们都觉得是多此一举，对吧？之后吵吵闹闹分分合合，她越来越习惯拿分手来做借口，我但凡挽留得不及时，她就会疑神疑鬼，问我是不是不爱她了，是不是爱上别人。我被她弄得都快精神分裂了，结果有天晚上她哭着给我打电话，不算哭吧，就是讲着讲着抽抽搭搭的。她说她想着我在那么远的地方，不知道我会不会跟别的女生在一起，跟别的女生一起吃饭牵手，就算知道我们班那些女生都有男朋友也会不安心。她知道说分手不对，说分手不好，但是想到我可能不爱她她就忍不住拿分手来试我。她躲在阳台打的电话，我躲在厕所接的，两边都很静，耳边就是她断断续续的呜咽，想哭又忍着，一抽一抽的，我的心也跟着一抽一抽的。后来她室友告诉我，她那段时间有轻度抑郁症，安眠药吃了好多——对，都是为我。她也没让我知道，反正那之后我觉得特亏欠她。女人是用来宠的，不是拿来伤的。我听人家说这话的时候还笑，谁没事干去伤自己女人？可惜我懂得太晚了——每一次我不接电话不回短信不理她，都是在伤她，在伤我们的感情，感情不是输给时间和距离，恰恰输给这些细微和琐碎……哎哟哟，弟妹来了啊，你小子福气真好，女朋友这么漂亮……不夸张不夸张，我兄弟一坐下就说你的好……没事没事，坐一小会，我也快讲完了。

分手？没有，我们谁都没有说分手。我记得我最后一次见她是大三那年，她穿着一件蓝衬衫，大白裙裙摆很大，拎着行李箱跟我说，爸妈让她去美国进修。我无条件地支持她。那个时候我们的感情很稳定了，她对我百分百的信任，偶尔因为哪个女的吃点小醋，也就是梗着脖子半天不理我，到晚上就又发条短信问我晚饭吃了些什么。我送她到安检口，站在外面看着她拉着我给她挑的哆啦A梦的旅行箱，她曾经跟我说，我就是她的哆啦A梦……她没有，没有爱上别人，真的没有……没哭，我没哭，哎呀老板你家羊肉串太辣了……没什么，我喝口酒……来，你把这封结婚请柬打开，对，打开。

没错，我的名字，和她的名字，你看这个“宇”的宝盖头是连着下边的字的。我一打开我就懵了，那天下午坐房间里就坐着，不干什么事，就是想她，然后就是哭……现在流的这叫汗，汗……唉，这张结婚请柬吧，是她姐结婚的时候印请柬的时候她偷偷拿出来的，我们俩高考之后到厦门去玩，她把这张结婚请柬写了我们俩的名字，用慢递寄三年寄给我。她说，如果到时候新娘不是她，她就来抢婚。她怕我们走不过三年……你别说，我现在特想结婚，真的，然后在婚礼上，等着她来把我抢走……不，没哭没哭，我真没哭，真的，真的是羊肉串太辣了，太辣……谢谢弟妹啊，我不是擦眼泪，我，我，我擦汗呐……就那天，她坐的那个航班的飞机失事了，整架飞机没一个人活下来的……我是她的哆啦A梦有什么用，我倒是想钻到地府，问阎王爷把她要回来。可是，可是……

兄弟，弟妹啊，你们俩都在呐我就说几句，趁着爱的人还在身边就好好爱吧。我最后悔的是没有回她的十五条短信，没有接她的二十七个电话，没有在她难受的时候陪在她身边。我怎么就没好好疼她，好好宠她呢？她说从美国回来我们就去澳大利亚看袋鼠，去南极看企鹅，去好多好多地方……弟妹啊，你们在一起这么久也挺不容易的，听我说，别留什么遗憾。爱最难圆满的就是遗憾，在失去之后，心上还有一个小缺口，那个缺口棱角分明，磨得你疼的呀……没事，我也知道她不在了。别说我多嘴，兄弟弟妹啊，如果，把你们在一起的每一天都当成最后一天，你们会一直幸福下去的……好了好了，酒喝完了，肉也吃完了，话也说到这，我就先走了……哟哟对，这个我还是带着吧，那什么你们俩什么时候结婚记得给我发个请柬，回见！
