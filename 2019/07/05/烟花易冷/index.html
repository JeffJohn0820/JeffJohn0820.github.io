烟花易冷

Jul 5, 2019

楔子
我在等，等一个不会再回来的人。
我在找，找那个等了我一生的人。

贰
一个月后，碧月到那个照相馆找人的时候，发现它被贴上了衙门的封条。她站在门口，看着车马川流，不知所措。
“诶……是你？”
碧月打量了一下来人，确定自己不认识他，还没开口，那人从外衣口袋里掏出一个牛皮纸袋：“这是你要的照片吧，兆铭交代我今天一定得来看一下，幸好你来了。”他将纸袋交给碧月，说了几句，抬脚要走，碧月下意识出声唤住他，咬了咬嘴唇，问：“他……呢？”
来人愣了一下，似是在思忖该不该告诉她，最后朝碧月的方向挪了几步。
“刺杀摄政王失败，他被关进大牢了。”
碧月知道前些天三叔遇刺的事，可不知道是他。她还记得阿玛跟福晋谈起时提到那个刺客名叫章兆铭。
碧月抽出照片看了一眼，不自觉握住胸前的怀表。额娘，他将女儿照得真漂亮，真像额娘。
章兆铭，就是他的名字么？碧月站在牢房外，试探地喊了一声。
章兆铭抬头看见是她，满脸惊讶，脱口问道：“你怎么在这？”再一想，“不对，你怎么进来的？”
陪着碧月来的嬷嬷替碧月回答道：“这是咱们肃亲王家的碧月格格。”
章兆铭瞪大了眼：“格格？这么说你跟他们是一伙的？”碧月一脸迷茫，章兆铭反应过来，苦笑道：“我怎么忘了，你们只是挂了旗人的名，很多事情是怪不得你们的。”
碧月将嬷嬷支开，蹲下来问他：“你为什么要刺杀三叔……就是……摄政王？革命党都是乱杀人的么？”
章兆铭道：“乱杀人？并不是的！你看看，你看看朝廷那些人，吃着民脂民膏，用着民脂民膏，到头来帮着洋人剥削搜刮民脂民膏，这种吸血鬼朝廷，还要它做什么？”章兆铭越说越大声，最后直接站起来了。碧月听他说的那些大逆不道的话，急忙做噤声的手势，压低了声音道：“你不要命啦？”
“章某自进同盟会效力孙先生以来，这条命就不是章某的了，是同盟会的，是四万万中国民众的。”章兆铭直直地看着碧月，眼神里是坚定和无畏。他拿过一旁桌上的纸笔，挥毫而就。

“衔石成痴绝，沧波万里愁。
孤飞终不倦，羞逐海鸥浮。
姹紫嫣红色，从知渲染难。
他时好花发，认取血痕斑。
慷慨歌燕市，从容作楚囚。
引刀成一快，不负少年头。
留得心魂在，残躯付劫灰。
青磷光不灭，夜夜照燕台。”

引刀成一快，不负少年头。碧月接过纸，将这两句默念了几遍，心中暗暗下了一个决定。
“你等我，我一定……一定会救你出去的。”碧月将颈上的怀表摘下来塞给章兆铭，匆匆而去。
章兆铭都来不及说些什么，珐琅掐丝的怀表躺在掌心，还带着少女的体温，颜色掉了些，不太鲜艳了，是很老的东西。打开，是一个旗装女子抱着一个粉琢玉砌的小孩子，孩子眼睛睁得大大的，煞是可爱，像极了方才信誓旦旦的少女。

叁

“阿玛，女儿要嫁给章兆铭。”碧月闯进福晋的厢房，她知道每天这个时候阿玛会在这里与福晋饮茶。见她闯进来，福晋先把脸拉下来，冲着看门的小丫头道：“你是怎么看门的？格格进门来也不通报？”便是说碧月不懂规矩了，碧月心里念着章兆铭，也不在乎，只是跪在肃亲王面前，坚定地重复了一遍：“阿玛，女儿要嫁给章兆铭。”
肃亲王这回听清楚了，手里茶盏却没捧稳，溅出几点茶水来。“你说哪个章兆铭？”他只知道一个章兆铭，那个大逆不道的乱党，那个在大牢里等死的家伙。
“就是几日前刺杀三叔的那个。”碧月毫不闪躲，“他现在在牢房里。”
“胡闹！”这回肃亲王气得把手里的茶盏连茶托都摔了，瓷片和茶水在碧月膝边炸开，碧月感觉到茶水的滚烫，没有挪动。
“阿玛，女儿同他已私定终身。信物便是额娘留下的怀表。”
肃亲王定睛一看，果然，碧月向来不离身的珐琅掐丝怀表不见了。提到先福晋，站在一旁的福晋脸色有些难看。肃亲王看着碧月，碧月也不闪躲，直直对上他的目光。
哀求，坚定，决绝。
肃亲王忽然发现，女儿真的是长大了，而那个人……也的确是去了啊……
“额娘如果还在，一定希望女儿嫁给自己喜欢的人。”碧月轻声道，许是因为想起了娘亲，眼角些微湿润。肃亲王沉默半晌，或许也想起了发妻在时的光景，叹了口气，伸手将碧月扶起来。
“我会尽力救他出来的。只是碧月你要记住，女儿家嫁了人……”
“嫁鸡随鸡，嫁狗随狗，碧月知道。”碧月鼻头一酸，她知道，自她出口要嫁给章兆铭时起，自己便要走上一条不能回头的路。
“去吧。”肃亲王仿佛一下子苍老一般，摆摆手，“去准备嫁衣吧。福晋，你替碧月准备嫁妆吧。”
碧月朝着肃亲王跪下，磕了三个响头，后退着出门去了。
“王爷……”
“好好准备，那是月儿的大事。”肃亲王放空了目光向远方望，似是陷入了回忆，“月儿她娘也会开心的啊……”十年生死两茫茫，纵使身边早有替代的温暖，回忆已是心口的一部分，又如何割舍？
一朝阶下囚，一朝度春宵。章兆铭手里攥着那块怀表，被人推搡着进了新房还觉得一切犹在梦中。三天前肃亲王上书为他和碧月格格求赐婚，而乱党之流行刺一事，以他“少不更事，受人蒙蔽”轻描淡写带过。
新房是在城郊的一栋宅子，肃亲王替碧月置办的，章兆铭什么都不用准备，只是穿着喜服。现在，新房里只有他们两个人，碧月在鸳鸯戏水的盖头底下被自己呼吸的热气熏得脸像被火烧，双手在喜服下面不安地交握，房间里静得两人的心跳声都清晰可闻。
“我去书房睡。”等了半天，等到章兆铭说了这句话，碧月心里一急，她看不见章兆铭的动作，只知道他要出门，直起身子：“诶……”
“哦，对了，这个还给你。”他走回来，从怀中掏出一个物件。
手心被塞进一个温热的扁扁的圆形物体，物体表面是熟悉的花纹，不熟悉的气息。
“傻姑娘，以后不要这么随便地将这个交出去了。”心也是如此。章兆铭张了张嘴，终是没说出后半句。他说完便抱着被子离开了，碧月甚至都来不及回一句——我并没有随便交出去。两面之缘就情根深种很荒谬吗？不是的。白蛇同许仙，撇开前世缘分不讲，也只是见了两面罢。
一个人在恰好的时间恰好入了另一个人的心，这是不能解释的，姑且称之为——宿命。

肆
额娘，女儿长大了，是不是很像额娘啊？
额娘，女儿嫁人了，就是给女儿拍照的那个人。
额娘，可不可以告诉女儿，怎样让那个人喜欢上女儿？
额娘……
碧月将照片和纸钱冥镪一点一点扔进火盆，看着跳跃的火苗舔舐着纸张，瞬间蜷曲变黑，风一吹，都散了。碧月坐在后院的小花园里，院子里没有多少伺候的人，一方面是福晋不想太破费。再者，碧月也有自己的考虑。她想，章兆铭如此痛恨皇族，那这个身份附带的权势地位排场等等，是不是也不喜欢？
看着火苗忽闪迷离，碧月有些恍神，等到章兆铭冲过来把她的手拉开，吼道：“你干什么？！”的时候，碧月才发现自己不知不觉向着那个火盆伸出了手。
“我……我……”碧月本就不善言辞，在王府的时候看福晋脸色行事做人更加沉默寡言，章兆铭这一吼，她一时也不知道该如何解释，“我”了半天也没“我”出什么来，最后只是指着那个火盆，轻声道：“今日是我额娘的忌日。”
章兆铭看到火盆里烧了一半的照片，心里一动：“你拍照是为了……”
碧月点点头，算是默认。章兆铭松开她的手，方才进院子就看见她把手伸过去，情急之下用的力气大了些，碧月的手腕明显出现一道红印。
“今日上元节，京城有花灯会的。你难得回来，能不能陪我去……”碧月问。自成婚以来，章兆铭经常不在家里，碧月隐隐觉得他在做一些可能会惊天动地，甚至令朝廷翻天覆地的大事。摄政王肃亲王都不止一次地提示碧月多注意点章兆铭动向，碧月每次都替他遮掩。说他安安分分，并无异常。
本欲出口的回绝的话语对上少女真诚期待又不安惶恐的眼神，出口便成了：“好。”
京城的花灯节很热闹，纵使穷途末路，这些耽溺享乐的人也从来不会亏待自己。外头大沽口炮火连天，外国人沿着铁路线向京城进发，可这座围城里的人，仍在享受自欺欺人的快乐。

碧月跟在章兆铭身后四处张望，目光落到一把琴上就再也移不开了。
“嗯？古筝？”
碧月点点头，目光温柔地盯着那把古筝。
那把古筝名为寂桐，曾是肃亲王为碧月的额娘做的的。当年的他们缘起这把古筝，“曲有误，周郎顾。”一顾一笑一倾心，便再也没能逃脱。只可惜，山盟海誓也只是曾经，肃亲王新纳的福晋不爱弹古筝，便将寂桐随随便便处理了。当年的承诺——不提也罢，或许承诺这东西之所以要许，只因为世间太多变数。
“梧桐半死清霜后，头白鸳鸯失伴飞。或许这把筝叫做寂桐，便已经注定了。”
是啊，注定，这世间最难抗争也最无可奈何的，便是这两个字了吧。
碧月兴致寥寥地回到家中，回到房间——章兆铭向来是睡在书房的。她坐到床边，却听见断断续续传来古筝的声音。碧月微微愣了一下，起身开门到院子里。
章兆铭端坐树下，面前是那把寂桐，见她出来，筝声戛然而止。
“今日不是你母亲忌日么？我想，她应该很开心听到这把筝的声音。”
远处天空一下又一下绽开华丽的烟花，升起，燃烧到极致的艳丽，美得那么不真实。碧月想说些什么，却发现没什么可以出口的，最后只是走过去，坐到他身旁，静静听他弹完一曲古筝。

伍
“那后来呢？”小女孩穿着宽大的尼姑服，两手托腮追问道，“后来他们是不是永远在一起幸福地生活下去了？”
她拨着灯花的手顿了顿，笑着拍拍小女孩的肩膀：“夜深了，快去睡吧。”
小女孩嘟着嘴，意犹未尽地离开了。她颤巍巍地站起来，推开窗，窗外不知何时又开始下雨，窗前摆着一把古筝，纤尘未染。
世上的事，哪里有那么多后来。
后来，章兆铭用那把第一次刺杀未竟的勃朗宁再次暗杀摄政王，这一次，子弹稳稳地飞进摄政王的心口，摄政王当场毙命，章兆铭不知所踪——他从未放弃过他“不负少年头”的抱负。他曾经问过，如果他走了，碧月会怎么办？碧月只是看着窗外下着的雨，笑了笑，说：“你走了，我就等你回来啊。”禁军队浩浩荡荡冲到她家抓人的时候，她还在磕磕绊绊地学着弹那首他弹过的曲子。摄政王亲眷及一众皇室族人都认为她是勾结乱党的叛徒，肃亲王站出来，极力证明她的下嫁是为了监视和招安章兆铭。力保之下，她也没有回到肃亲王府，在一个雨夜，背着寂桐，上了后山，入了寺庙，落尽红尘。
她有时会望着寂桐，就想起那个人。岁月渐渐在木鱼声中流去，年华亦是不动声色地离开，在这样下着雨的静夜，她也会问自己，为什么？
为什么能够数面之缘就情根深种？
为什么愿意终此一生地无悔等待？
而她自己也给不出回答。
又过了几年，她已经老得走不动了，给小徒弟讲的故事也总是停在那个后来，也就没有了后来。躺在床上，小徒弟把寂桐抱到她面前，她抬手拨了一下，似乎弦上仍有那人指尖的温度。
你走了，我在等你回来——可是啊，我把我这一生都等完了，你怎么还不回来呢？
其实，哪里来的那么多为什么？汤显祖早就说过，情不知所起，一往而深而已。爱情同战争一样，讲求天时地利，那个人，只是在最恰好的时候入了心，就此，无可救药。她已经忘了那人的模样，忘了他曾说过的话，却依旧清晰地记得漫天的烟花盛放到极致的绚烂，和树下那曲古筝——那也就够了。
便如曲终人散，烟花易冷，掌心留过的余温已够温暖余生。
后来，听说有一个拿着块怀表和一张照片来找人的男人。
他说，他把他的妻子弄丢了。
他说，当年仓促离开的时候，不知为何带走了这两样东西。
他说，她还在身边的时候，他眼里只有天下。然爱之弄人，在于离开之后方看得清，殊不知缘断便断，轻如蛛丝。
他说，他半生戎马打天下，遇到叫“璧君”“寒月”的女子，总会愣神。
他说，他的妻子，叫碧月。

陆
山上的木鱼一声一声伴着山下长流的细水。庵庙里的主人换成了年轻的女尼，她的身边也跟着一个小女孩，拖着不合身的尼姑服听她讲一个没有结局的故事。
山下的男人拿着怀表和照片，步履蹒跚地穿梭于大街小巷，怀表的表面已十分光滑，时针也不走了，照片被摩挲得模糊了眉目。他在问，问他们知不知道那个叫碧月的女子。
她说，会一直等我回来的。
